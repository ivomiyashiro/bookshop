## 🛠 Installation

```bash
# clone repository
$ git clone https://github.com/ivomiyashiro/bookshop-api.git

# install dependencies
$ pnpm install
```

## 💡 The .env file

```
BASE_API_URL_PROD=
BASE_API_URL_DEV=
BASE_URL_DEV=
BASE_URL_PROD=
```

## 🚀 Running the app

```bash
# development
$ pnpm run dev

# production mode
$ pnpm start
```

## 🧪 Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
