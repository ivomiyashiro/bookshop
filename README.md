## 🛠 Installation

```bash
$ npm install
```

## 🚀 Running the app

```bash
# development
$ npm run dev

# production mode
$ npm start
```

## 🧪 Test

```bash

$ npm run test

# e2e tests

# test coverage
$ npm run test:cov
```
